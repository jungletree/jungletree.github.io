TOTEM PIPE Screenshots:

TOTEM PIPE

TOTEM PIPE is an agile puzzle 3D games. Players control a Japanese-style puppet in a slowly rising tower, along the way to keep collecting the hidden scroll to make up the ladders, finally find the path to the top of totem pole.

The level was designed carefully. There is no one scroll more or less, do not worry about wast them like bullets. Aways walk ahead is good idea, but sometimes you will find you need to walk backward when you run out of scrolls. The same as other running or racing games, the time is always limited. If you are time-out, you are killed. Then you are lost in the fog. So brave heros, come and give it a try!

How to Play:

  • Move : joy stick of the left screen
  • X button : build ladders
  • Z button : eat or destroy the platform (something marked with “Z” character), so that you could make a way backward.

勇闯天塔

勇闯天塔是一款3D类的敏捷解谜类小游戏。游戏中玩家控制一个日本式的木偶，在一座徐徐上升的天塔之中，沿途不断收集塔内隐藏的卷轴制制作往上的梯子，并试图找出通向塔顶的路径。

游戏中的关卡是精心设计的。游戏中所需要的卷轴是恰到好处的安排的，它们不像是射击类游戏中可以随意射出的子弹一样没有数量上的限制。沿着往上的梯子往上走总是没有错的，但是某些时候当你的卷轴用光了的时候迂回前进也是明智的。就像其他的跑酷类或者赛车类游戏一样，游戏的时间也是有限的。一旦你超时了，那么你就会迷失在迷雾之中，然后就挂掉了。勇敢的骚年，来挑战一下吧！

游戏玩法:

  • 移动： 屏幕左边虚拟摇杆
  • X 按钮：制作梯子
  • Z 按钮: 吃掉或者销毁平台（就是那些上面标记为字符“Z”的东东），从而开辟向下的道路
