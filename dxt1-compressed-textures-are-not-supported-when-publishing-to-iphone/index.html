Solution:

  1. Go to:

on MAC
Users/%userprofile%/Library/Unity/cache/packages/packages.unity.com/com.unity.ads
@1.0.7/Editor/Resources/Editor/

on Windows
%userprofile%\AppData\LocalLow\Unity\cache\packages\packages.unity.com\com.unity.ads
@1.0.7\Editor\Resources\Editor

  1. Open portrait.jpg.meta and landscape.jpg.meta
  2. Find field textureFormat: -1 (or 1)
  3. Change this value to this one textureFormat: 4

By the way, you need to check whether there are some .dds textures under the Assets folder while building the iPhone version. If there are some, you need to change it to .png file or just remove it (if possible).
